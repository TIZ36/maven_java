# maven_java
maven_java project template

## 1. 项目结构
```
├── README.md
├── pom.xml
├── src
│   ├── main
│   │   ├── java
│   │   │   └── com
│   │   │       └── pandio
│   │   │           └── app
│   │   │               └── App.java
│   │   │               └── SomeJavaClass.java
│   │   └── resources
│   │       └── log4j.properties
│   └── test
│       ├── java
│       │   └── com
│       │       └── example
│       │           └── demo
│       │               └── AppTest.java
│       └── resources

```

1. 环境变量里有mvn命令
2. 在pom.xml同级目录用命令 mvn install
3. target目录下生成jar包, 使用 xxxx-jar-with-dependencies.jar
